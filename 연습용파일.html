HTML

자세한건 요걸루v

HTML ( Hypertext Markup Language )은 웹 페이지 및 웹 응용 프로그램 을 만들기위한 표준 마크 업 언어 입니다.
함께 캐스 케이 딩 스타일 시트 (CSS)와 자바 스크립트 , 그것은을위한 초석 기술의 화음 형성 월드 와이드 웹을,
웹 브라우저는 웹 서버 또는 로컬 저장소 에서 HTML 문서를 받아 문서를 멀티미디어 웹 페이지로 렌더링 합니다.
HTML은 웹 페이지의 구조를 의미 론적으로 설명 하고 원래 문서의 모양에 대한 단서를 포함합니다.
HTML 요소 는 HTML 페이지의 구성 요소 입니
HTML 구문을 사용하면 이미지 및 대화 형 양식 과 같은 다른 객체 가 렌더링 된 페이지에 포함될 수 있습니다.
HTML은 제목, 단락, 목록, 링크, 따옴표 및 기타 항목과 같은 텍스트의 구조적 의미 를 나타내서 구조화 된 문서 를 만드는 방법을 제공 합니다.
HTML 요소는 꺽쇠 괄호를 사용하여 작성된 태그 로 구분 됩니다.

태그는 같은 과 직접 페이지에 내용을 소개합니다.
다른 태그 (예 :img/input/p
서라운드를 제공하고 문서 텍스트에 대한 정보를 제공하며 다른 태그를 하위 요소로 포함 할 수 있습니다.
브라우저는 HTML 태그를 표시하지 않지만이를 사용하여 페이지의 내용을 해석합니다.
