CSS

자세한 설명은 여기에v

캐스 케이 딩 스타일 시트 (CSS)는 인 스타일 시트 언어 묘화에 사용되는
프리젠테이션 에 작성하는 문서의 마크 업 언어 와 같은 HTML을 말합니다.
CSS는 HTML 및 JavaScript 와 함께 World Wide Web 의 기본 기술입니다.
CSS는 레이아웃 , 색상 및 글꼴을 포함하여 프레젠테이션과 콘텐츠를 분리 할 수 ​​있도록 설계되었습니다.
이러한 분리는 콘텐츠를 향상시킬 수있는 접근성 , 프리젠 테이션 특성의 사양에 더 많은 유연성과 제어를 제공,
여러 수 있도록 웹 페이지를 별도의 .css 파일에 관련 CSS를 지정하여 서식을 공유하고,
구조적 내용의 복잡성과 반복을 감소하였습니다.
서식과 내용을 분리하면 화면 상, 인쇄 중, 음성 (음성 기반 브라우저 또는 스크린 리더를 통해 ),
점자 기반 (Braille-based ) 과 같은 다른 렌더링 방법에 대해 서로 다른 스타일의 동일한 마크 업 페이지를 제공 할 수도 있습니다.
촉각 장치, CSS에는 모바일 장치 에서 콘텐츠에 액세스하는 경우 대체 서식 지정 규칙이 있습니다 .
